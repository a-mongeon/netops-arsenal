# KeePass

## Usefull tips

### Ouvrir SDS depuis KeePass
Ajouter cette commande dans dans le champ URL de l'entrée :
```
cmd://"C:\Program Files\Arkoon\Security BOX\Kernel\SBCMD.EXE" /U -p {PASSWORD}
```
### Ouvrir un autre KeePass depuis KeePass
Ajouter cette commande dans dans le champ URL de l'entrée :
```
cmd://"C:\Program Files\KeePass Password Safe 2\KeePass.exe" "database.kdbx" -pw:{PASSWORD}
```
