# Manage Simple Storage Service (S3)

## Useful commands with aws-cli
Lister les buckets :
```
aws --profile <profile> s3 ls
```
Créer un bucket
```
aws --profile <profile> s3 mb s3://<bucket>
```
Supprimer de manière récursive tous les objets d'un bucket :
```
aws --profile <profile> s3 rm s3://<bucket> --recursive
```
